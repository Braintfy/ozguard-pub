## Актуальная версия

**v5.9.26** — стабильнее работает с большими пакетами жалоб, крупными доказательствами и повторными запросами файлов в сценарии «Мой контент».

## Что делает OZGuard

Вы вводите артикулы (SKU) своих товаров — расширение автоматически проверяет карточки на OZON и собирает список продавцов: SKU, названия магазинов и цены. После сбора можно перенести найденные SKU в раздел жалоб и запустить обработку через чат поддержки seller.ozon.ru.

Основные возможности:
- поиск конкурентов по списку SKU;
- экспорт результатов и история сессий;
- импорт SKU из Excel-файла OZON;
- исключения для своих магазинов при сборе и подаче жалоб;
- автоматизация жалоб в режимах «Плагиат карточек», «Мой контент» и «Мой бренд»;
- сохранение доказательств в текущей установке расширения;
- поддержка общих файлов и файлов, привязанных к конкретным SKU.

## Установка

1. Скачайте или клонируйте этот репозиторий
2. Откройте `chrome://extensions/` в браузере
3. Включите **Режим разработчика** (переключатель в правом верхнем углу)
4. Нажмите **Загрузить распакованное расширение**
5. Выберите папку с файлами расширения
6. Иконка OZGuard появится на панели расширений

Важно: если распакованную версию загрузить из новой папки, Chrome может считать её новым расширением. В таком случае старые локальные доказательства и настройки будут недоступны — загрузите файлы заново.

## Как пользоваться

### Сканирование

1. Нажмите на иконку OZGuard в панели расширений
2. Введите SKU товаров (по одному на строку) в поле ввода
3. Нажмите **Найти конкурентов**
4. Расширение откроет окно OZON и автоматически проверит каждый товар
5. Результаты появятся в списке: название товара, количество конкурентов, их SKU и цены

**Кнопки в результатах:**
- **Скопировать** — полный текст со всеми данными
- **Только SKU** — список SKU конкурентов
- **Excel** — скачать CSV-файл для Excel
- **В жалобы** — отправить SKU конкурентов в раздел Жалобы

### Пакетный импорт (PRO)

1. В личном кабинете OZON: **Товары → Цены товаров → Скачать шаблон**
2. Перетащите скачанный `.xlsx` файл в зону загрузки в расширении
3. Выберите статусы товаров (Продается / Готов к продаже / Не продается)
4. Нажмите **Применить** — SKU попадут в поле ввода

### История

Последние 10 сессий сканирования сохраняются автоматически. Для каждой сессии доступно:
- Скачать Excel
- Посмотреть детали
- Отправить SKU конкурентов в Жалобы

### Жалобы (PRO)

1. Добавьте SKU конкурентов — вручную, кнопкой **В жалобы** или через **Импорт из Excel**
2. Откройте [seller.ozon.ru](https://seller.ozon.ru/app/messenger/?group=support_v2) в браузере
3. Выберите режим жалобы: **Плагиат карточек**, **Мой контент** или **Мой бренд**
4. Прикрепите доказательства: общие файлы для всего пакета или отдельные файлы для конкретных SKU
5. Нажмите **Начать** — бот автоматически обработает очередь

Что учтено в последних версиях:
- большие пакеты запускаются без передачи тяжёлых base64-файлов в стартовом сообщении;
- крупные файлы читаются и прикрепляются только в момент обработки нужного SKU;
- для больших видео и документов увеличено окно проверки появления файла в чате;
- если Ozon повторно просит доказательства в сценарии «Мой контент», OZGuard делает 2 контрольных повтора последнего файла, затем помечает только текущий SKU ошибкой и продолжает пакет;
- антибот-паузы работают автоматически, без ручного лимит-гейта после фиксированного числа обращений.

### Настройки

- **Лицензия PRO** — введите код активации или попробуйте бесплатно 7 дней
- **Исключения** — названия своих магазинов, которые не попадут в результаты
- **Жалобы: магазины-исключения** — магазины, на которые не нужно отправлять жалобы
- **Доказательства** — общие и привязанные к SKU файлы для жалоб
- **Задержка** — пауза между запросами (рекомендация: 2000–3000 мс для 5–20 SKU)
