📖 Инструкция OZGuard
Что такое OZGuard?

OZGuard находит всех конкурентов, которые продают те же товары что и вы на OZON. Вы вводите SKU своих товаров — расширение по каждому запрашивает данные карточки, собирает список всех продавцов и показывает их цены. После этого можно одним кликом отправить нарушителей на проверку через жалобы.

Сайт: codefic.ru

🔍 Сканирование
1
Введите SKU своих товаров в поле (по одному на строку). SKU — это числовой артикул товара на OZON, его видно в адресной строке карточки товара или в личном кабинете продавца.
2
Нажмите «Найти конкурентов» — сбор идёт прямыми API-запросами в фоне, без открытия окон. Нужна открытая вкладка ozon.ru с авторизацией (если её нет — расширение откроет фоновую само).
3
В Результатах вы увидите по каждому товару: название, количество конкурентов, их SKU и цены. Кнопка ⎘ рядом с вашим SKU копирует его. Ссылка «открыть на OZON» ведёт на карточку. Кнопка ⎘ рядом с «N конк.» копирует сразу все SKU конкурентов по этому товару.
4
Кнопка 📎 К доказательствам рядом с каждым вашим SKU переносит его в Настройки → блок «Доказательства для отдельных SKU». Если к SKU уже привязаны файлы — кнопка меняется на ✓ Доказательства зелёным. Так удобно за один проход по результатам собрать все доказательства.
⏸ Пауза и ⏹ Стоп работают прямо во время сбора. Скан продолжается в фоне, даже если закрыть popup — откройте его снова, прогресс восстановится.
📦 Пакетный импорт XLSX
1
В личном кабинете OZON перейдите: Товары → Цены товаров → Скачать шаблон.
2
Перетащите скачанный .xlsx файл в зону «Загрузите XLSX шаблон» или нажмите на неё.
3
Выберите нужные статусы товаров — «Продается», «Готов к продаже», «Не продается» — и нажмите «Применить». SKU попадут в поле ввода.
По умолчанию отмечен только «Продается» — чтобы не тратить время на снятые товары. Кнопка 🕘 Последние быстро возвращает 3 последних применённых XLSX-списка.

⚙️ Настройки
  • Лицензия PRO — нужна только для автоматической подачи жалоб. Весь функционал сбора SKU и пакетного импорта XLSX бесплатен. Купить PRO: codefic.ru/#pricing
  • Исключения при сборе — названия своих магазинов, которые не попадут в результаты. По одному на строку, частичное совпадение. Переключатель CN включает строгий режим: в результатах остаются только продавцы с явной меткой CN в реквизитах Ozon; продавцы без подтверждённой метки не собираются.
  • Задержка — пауза между запросами к OZON (мс). Рекомендация: 2000–3000 при 5–20 SKU, 4000–5000 при 50+ SKU.
  • Подсказки интерфейса — кнопки «Скрыть все подсказки» / «Показать все подсказки» управляют баннерами во вкладке «Жалобы» (VPN-предупреждение, инструкция).
  • Жалобы: антибот-паузы — при больших пакетах бот продолжает работу автоматически и делает паузы между обращениями без ручного подтверждения.
  • Жалобы: доказательства для отдельных SKU — добавьте свой SKU и файлы (до 45 MB каждый: видео, сертификаты, PDF). Бот при подаче жалобы на конкурента узнаёт из какого вашего SKU он был найден и прикрепит именно эти файлы. Если у вас несколько своих SKU указали на одного конкурента — файлы объединяются. Если для SKU нет привязанных файлов — используется общий пул из вкладки «Жалобы». Если и общего нет — SKU помечается failed.
📨 Жалобы PRO
1
Введите SKU конкурентов вручную, нажмите «В жалобы» из Сканирования, или загрузите Excel-файл кнопкой «Импорт из Excel».
2
Откройте seller.ozon.ru → Поддержка в браузере (вкладка должна быть открыта).
3
Выберите тип жалобы — три варианта:
  • Плагиат моих карточек — стабильный путь через «Личный кабинет → Кабинет бренда → Жалоба → Плагиат → Использование моих фото/видео/текста». Используется чаще всего.
  • Мой контент — путь через «Товары и Цены → Контроль качества → Нарушение правил площадки → Использование моих фото, видео, текста».
  • Мой бренд — тот же путь, но с финальной кнопкой «Использование моего бренда».
При 5 ошибках подряд бот сделает паузу и попросит подтверждения — это защита от изменившегося интерфейса Ozon.
4
Нажмите «📨 Начать». Бот сам откроет чат поддержки, пройдёт дерево меню выбранного типа, введёт каждый артикул и прикрепит файлы-доказательства (если загружены).
🧪 Тест — включите перед первым прогоном на 2–3 SKU: при локальной ошибке бот поставит пакет на паузу для проверки. Жалобы при этом реальные. После успешной проверки выключите «Тест» для большого или ночного пакета; критические защиты продолжат работать.
📎 Если для SKU настроено несколько доказательств, бот отправляет их по порядку и перед каждым следующим файлом заново проверяет активный чат. Неопределённый черновик, смена треда или шага не разрешают повторную отправку.
📎 Доказательства — два режима:
Общий пул — файлы из зоны «Доказательства» во вкладке «Жалобы». Прикрепляются ко ВСЕМ жалобам.
Per-SKU — файлы из блока «Доказательства для отдельных SKU» в Настройках. Привязываются к конкретному вашему SKU и прикрепляются только к нарушителям, которые были найдены по нему. Приоритет выше общего пула.
Файлы до 45 MB сохраняются в браузере и переживают закрытие popup и перезагрузку.
🛎 Блок «Проблемные SKU» — показывает три категории:
Передано оператору — обращение ушло на ручное рассмотрение (не ошибка, Ozon эскалировал);
Без нарушений — Ozon проверил обращение, но не подтвердил нарушение;
Ошибки — failed по техническим причинам (нет файлов, поле ввода недоступно, зацикливание).
Для каждой категории: 📋 Копировать, ↻ В очередь (для повтора), 📥 XLSX, ✕ Очистить.
⏱ Большие пакеты — бот не останавливается на лимите обращений, но сохраняет антибот-паузы между действиями. Остановить или поставить на паузу можно вручную.
❓ Частые вопросы
  • «Товар не найден или не в наличии» — OZON вернул категорию вместо карточки (товар снят с продажи). Это нормально — расширение пропускает такие SKU.
  • «OZON API не ответил» по многим SKU? — нажмите «🔌 Проверить OZON API» в Настройках: подскажет причину (нет авторизации, антибот, устаревшая вкладка). Чаще всего помогает обновить страницу ozon.ru (F5) и войти в аккаунт.
  • Бот завис? — нажмите ⏹ Стоп и запустите снова.
  • Не запускаются жалобы? — убедитесь что вкладка seller.ozon.ru открыта в браузере.
  • Ложный failed у файла? — если бот показал failed, но файл в чате реально есть и Ozon ответил «Скрыли товар» — это было в старых версиях. С v5.9.11 верификация смотрит и на ответ бота, и на факт прикрепления.
  • Нужна помощь? — codefic.ru
